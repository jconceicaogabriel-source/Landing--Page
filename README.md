# 🌐 Landing Page — Desenvolvimento Web

Projeto de landing page pessoal desenvolvido como prática na **EBAC**, apresentando serviços de desenvolvimento web com foco em boas práticas de HTML5 semântico.

---

## 👤 Sobre o Projeto

Landing page criada por mim, desenvolvedor web em formação, com o objetivo de aplicar os conceitos de semântica HTML5, acessibilidade e otimização para motores de busca.

---

## 🗂️ Seções da Página

| Seção | ID | Descrição |
|---|---|---|
| Navegação | — | Menu com links para Sobre, Serviços e Contato |
| Sobre Mim | `#sobre` | Apresentação pessoal e proposta de valor |
| Especialidades | `#servicos` | Serviços oferecidos |
| Contato | `#contato` | Formulário para solicitação de orçamento |
| Rodapé | — | Copyright e localização |

---

## 🛠️ Especialidades

- **Design Responsivo** — Sites adaptados para celulares, tablets e computadores
- **Otimização SEO** — Melhor ranqueamento nos motores de busca
- **Performance e Desempenho** — Carregamento rápido e código limpo seguindo padrões modernos

---

## 📬 Formulário de Contato

O formulário coleta as seguintes informações:

- **Nome** — Campo de texto obrigatório
- **Contato** — Número de telefone/WhatsApp obrigatório
- **Mensagem** — Área de texto livre
- **Ação** — Botão "Solicitar Orçamento"

---

## 🧰 Tecnologias Utilizadas

- **HTML5** — Estrutura semântica com uso de `<header>`, `<main>`, `<section>`, `<article>`, `<footer>`, `<address>`, `<fieldset>` e `<legend>`
- **VS Code** — Editor de código com extensão Go Live para visualização em tempo real

---

## ✅ Boas Práticas Aplicadas

- Semântica HTML5 para acessibilidade e SEO
- Uso correto de atributos `for`/`id` em formulários
- Atributo `required` nos campos obrigatórios
- Comentários descritivos ao longo do código
- Separação visual de seções com `<hr>`
- `lang="pt-BR"` definido na tag `<html>`
- Charset UTF-8 e viewport configurados no `<head>`
